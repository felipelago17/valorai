# How to Cite This Framework

_Last reviewed: 2026-06-27_

ValorAI is an open academic framework. If you use the taxonomy, methodology, evidence base, or worked examples in research or professional work, please cite it so readers can verify and build on the source material.

---

## BibTeX

```bibtex
@misc{lago_valorai_2026,
  author       = {Lago, Felipe},
  title        = {{ValorAI}: {A} Framework for Measuring the True Cost and
                  Realised Value of {AI}},
  year         = {2026},
  howpublished = {\url{https://felipelago17.github.io/valorai/}},
  note         = {Academic framework. Accessed: 2026}
}
```

## APA (7th edition)

Lago, F. (2026). *ValorAI: A framework for measuring the true cost and realised value of AI*. https://felipelago17.github.io/valorai/

## Chicago (Author–Date)

Lago, Felipe. 2026. "ValorAI: A Framework for Measuring the True Cost and Realised Value of AI." https://felipelago17.github.io/valorai/.

## Citing specific components

When citing a particular page rather than the framework as a whole, append the page title and path, for example:

> Lago, F. (2026). *Sector lenses* [in ValorAI]. https://felipelago17.github.io/valorai/sector-lenses/

> Lago, F. (2026). *Energy: Predictive maintenance (JV context)* [in ValorAI]. https://felipelago17.github.io/valorai/use-cases/energy-predictive-maintenance/

---

## Citing the primary sources

ValorAI is a synthesis framework, not a primary data source. When citing a specific quantitative finding (e.g. the 40–60% hidden-cost range, the 95% pilot-failure rate), cite the primary source from the [Evidence Bibliography](evidence/bibliography.md) or [Literature Bibliography](literature/bibliography.md), not ValorAI itself. ValorAI is the interpretive framework; the primary sources are the evidence.

---

## Epistemic note

Every quantitative figure in this framework is traceable to a primary source or marked `[VERIFY]`. If you encounter a claim without a source, please open an issue in the [repository](https://github.com/felipelago17/valorai) — that is a gap to be closed, not a feature.
