# Cross-Cutting Lens: Financial TCO & Value Sizing

This folder collects sources that operate at the aggregate or cross-sector level on the financial value-and-cost balance: top-down value-sizing studies, macro-productivity estimates, and ROI reality-checks. These sources calibrate the outer bounds of what AI deployments can realistically deliver, and anchor sector-level projections.

**Scoring across sectors:** Use value-sizing studies (McKinsey, Stanford HAI) as ceilings on the optimistic scenario. Use reality-check sources (Acemoglu 2024, MIT NANDA 2025) as empirical floors. Apply the ValorAI full-lifecycle TCO model to translate gross potential value into realistic net value.

## Sources filed here

- `mckinsey-2023-genai-economic-potential.md` — top-down sector value-sizing (value-positive)
- `stanfordhai-2025-ai-index.md` — annual time-series for cost curve and capability (mixed)
- `acemoglu-2024-simple-macroeconomics-ai.md` — conservative macro productivity ceiling (reality-check)
- `nanda-2025-genai-divide.md` — enterprise ROI reality-check; pilot-to-production gap (reality-check)
